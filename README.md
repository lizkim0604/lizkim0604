> **재무제표 기반 재무 이상징후 탐지기**
> 
- https://github.com/lizkim0604/dart-redflag
- **문제 정의**
    - 감사인의 효율성을 위해 전기대비 비정상적인 증감을 보이는 징후를 탐지한다면 효율적인 감사가 이루어질 것이라 판단함
- **작동 과정**
    1. DART Open API 사용하여 상장기업 재무제표 수집
    2. 4개의 룰을 적용하여 위험 신호 발견
        1. 증감률이 50% 이상이면서 변동 금액이 10억 이상
        2. 매출채권 증가율이 매출 증가율보다 20%p이상 큼
        3. 장부상 순이익은 흑자인데, 실제 영업활동으로 들어온 현금은 마이너스
        4. 재고자산 증가율이 매출 증가율보다 30%p 이상 큼
    3. 위험 신호에 대해서 Gemini API로 해석 및 절차 제안
    4. 리포트 생성
- **사용 기술**
    - Python, HTML, Gemini API
- **느낀 점**
    - 모든 것을 LLM API에 의존한다면 시간과 API 사용량이 과다해지기 때문에 직접 룰을 적용하여 스크리닝하는 것이 효율적임을 알 수 있었다.
